Lahzo design system

The shared visual language for Lahzo internal tools — built on Tailwind CSS v4 and daisyUI v5, styled with the Lahzo Light theme. Square corners, 1.5px borders, a muted earthy palette.

Version 1.1 · desktop internal tools · light theme only

Colors

Every surface and control references a theme token, never a raw value. On any tinted surface, text is base-content so contrast stays safe.

Icons

Phosphor regular, the only icon mechanism — <i class="ph ph-{name}">. Sizes 16 / 20 / 24 / 32, with 20 the default in controls. No emoji, no other sets.

Accessibility

Every text and background pair meets WCAG AA — 4.5:1 for normal text, 3:1 for large text and ui graphics.

Muted text is base-content at 0.8 opacity — that one value only.

State is never carried by color alone: pair it with an icon and a word.
